1. Introduction



"Video Screen" is a screensaver that displays videos. When the screensaver starts, it plays registered videos one after another. Additionally, the display position changes periodically.
"Video Screen" is freeware.

*Please be sure to read the precautions before use.

Supported Video Formats


Videos playable in "Video Screen" depend on your system environment.
Any video format playable in Windows Media Player can be played.

Supported OS


Windows 11

*Compatibility with Windows 10 and earlier is unconfirmed.

Precautions

  • Settings are not compatible with V2.xx or earlier (settings cannot be carried over).
  • Please observe the following regarding redistribution:
    　　・Do not modify the package or distribute only parts of it.

  • Suspend and Standby modes are not supported. The system may lock up.
    It is possible to avoid locking by configuring the settings to stop video playback before standby starts (available since v1.70).

2. Install / Uninstall

Installation

1) Copy VS.SCR to any folder.
2) Right-click VS.SCR and select "Install."

Uninstallation

Delete VS.SCR from the folder where it was installed (c:\Windows\System32).
*If you wish to perform a complete uninstallation, delete everything under the following folder:
　C:\Users\XXX\AppData\Local\BearBeetle

3. Settings


　[Note]
　　*"Video Content" has the same meaning as "Video File."

1. Open the Screensaver Settings screen.
Right-click on the desktop -> "Personalize" -> "Lock screen" -> (Related settings) "Screen saver"


2. Select "Video Screen" in the "Screen saver (S)" dropdown.


3. Click "Settings (T)..."
A dialog titled "Video Screen" will appear.
At this point, nothing is registered in the "Playback Content List."
　　　


4. Click "Add File" or "Add URL" to specify the video files you want to register.
　　　　*In "Add File," you can select multiple files at once by holding the Shift or Control key while selecting.
　　　　*"Add URL" is used to play video files from the internet.
　　However, services that do not specify a direct file, such as YouTube, cannot be played.
　　　


5. To remove an entry from the "Playback Content List,"
　　select the item and click "Delete."

6. To remove all entries from the "Playback Content List,"
　　select an item and click "Delete All."

7. Select an item from the "Playback Content List" and click "↑" to move it up one line.
　　Click "↓" to move it down one line.

8. Check "Random Playback" to play items in a random order.
　　　　*The same content may occasionally play twice in a row.

9. Check "Resume Playback" to resume from where you left off the next time the screensaver starts.

10. Check "Mute" to
　　silence the sound only while the video is playing (while the screensaver is active).

11. Check "Display Filename" to display the filename at the top of the screen
　　during video playback (while the screensaver is active).

12. "Display Size" allows you to set the size to 0.5x, 1x, 2x, or 3x the original size.
　　Also, if you set "Display Size" to "Custom," you can set the magnification linearly via "Display Size (%)."

13. "Change Position Interval" allows you to set the interval for changing the video display position between 1 and 600 seconds.
　　　　*Setting it to 0 seconds fixes the video position to the center.

14. "Playback Stop Time" allows you to set the interval until video playback ends between 1 and 600 minutes.
　　　　*Setting it to 0 minutes means video playback will not end until the screensaver itself terminates.
　　　　*If you enable "Suspend" or "Display Power Saving" functions, please ensure
　　　　 the settings are configured to stop video display before the power turns off.

15. Click "OK" to apply the settings.

16. While the screensaver is running, pressing the "N" key on the keyboard
　　will immediately move to the next video content. Pressing the "R" key
　　will return to the beginning of the current video content. Pressing "S"
　　toggles Mute ON and OFF. However, these states are not saved to the settings.
　　　*Pressing any other key will exit the screensaver.

17. Limiting the Display Range (Advanced Settings)
　　　C:\Users\XXX\AppData\Local\BearBeetle\VS2\VS2.INI
SCREEN_WIDTH=640
SCREEN_HEIGHT=480
Enter the desired width/height values in the blue text area (640/480) to limit the range.
